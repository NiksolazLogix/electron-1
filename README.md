# Install package

- with npm: `npm i`
- with yarn `yarn`

***

# Start project 

### with NPM
###### normal
`npm run start` 
###### and with hot reload
`npm run start:hot`

### with YARN
###### normal
`yarn start` 
###### and with hot reload
`yarn start:hot`
